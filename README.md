# Object-Oriented Car Rental System (Java)

A robust Command Line Interface (CLI) application built with Java to manage a car rental fleet. This project serves as the crown jewel of my portfolio, demonstrating my solid understanding of Object-Oriented Programming (OOP) core principles, data structures, and clean code architecture.

## 🚀 Features
* **Fleet Management:** Dynamic tracking of multiple vehicles using Java Collections (`ArrayList`).
* **Real-time Availability:** Vehicles change status dynamically between "Available" and "Rented".
* **Interactive CLI Menu:** A user-friendly console menu driven by input validation (`Scanner`).
* **Business Logic:** Prevents double-renting of vehicles and handles non-existent IDs gracefully.

## 🛠️ Tech Stack & Skills Demonstrated
* **Language:** Java (JDK)
* **Tools:** IntelliJ IDEA, Git/GitHub
* **OOP Concepts Demonstrated:**
    * **Encapsulation:** Using `private` fields with explicit getters and setters to protect data integrity.
    * **Object Interaction:** Separating data models (`Vehicle`) from controller logic (`RentalSystem`) and execution (`Main`).
    * **Collection Framework:** Managing dynamic data lists efficiently with `List` and `ArrayList`.
