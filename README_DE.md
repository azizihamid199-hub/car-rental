# Objektorientiertes Autovermietung-System (Java)

[🇬🇧 Switch to English Version](README.md)

Eine robuste Kommandozeilen-Anwendung (CLI) in Java zur Verwaltung einer Fahrzeugflotte. Dieses Projekt ist das Herzstück meines Portfolios und demonstriert mein solides Verständnis der Kernprinzipien der objektorientierten Programmierung (OOP), von Datenstrukturen und sauberer Code-Architektur.

## 🚀 Funktionen
* **Flottenmanagement:** Dynamische Verwaltung mehrerer Fahrzeuge mithilfe von Java Collections (`ArrayList`).
* **Echtzeit-Verfügbarkeit:** Fahrzeuge wechseln den Status dynamisch zwischen "Verfügbar" und "Vermietet".
* **Interaktives CLI-Menü:** Ein benutzerfreundliches Konsolenmenü, gesteuert durch Eingabevalidierung via `Scanner`.
* **Business-Logik:** Verhindert die Doppelvermietung von Fahrzeugen und fängt nicht existierende IDs sauber ab.

## 🛠️ Tech-Stack & gezeigte Fähigkeiten
* **Sprache:** Java (JDK)
* **Tools:** IntelliJ IDEA, Git/GitHub
* **Gezeigte OOP-Konzepte:**
    * **Datenkapselung (Encapsulation):** Nutzung von `private` Feldern mit expliziten Gettern und Settern zum Schutz der Datenintegrität.
    * **Objektinteraktion:** Saubere Trennung von Datenmodellen (`Vehicle`), Controller-Logik (`RentalSystem`) und Programmausführung (`Main`).
    * **Collection Framework:** Effiziente Verwaltung dynamischer Datenlisten mit `List` und `ArrayList`.
